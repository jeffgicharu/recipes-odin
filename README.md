# Landing Page Project

This project is a landing page built as part of the Foundations Course. It follows a clean, minimalistic design with multiple sections including a header, hero area, informational cards, testimonial, call-to-action, and footer. The project utilizes HTML and CSS along with Google Fonts to achieve the desired look and feel.

## Overview

This landing page is designed to closely follow a provided layout, which includes:

- **Header**: Contains a logo and navigation links.
- **Hero Section**: Features a bold heading, subtext, and a "Sign up" button with an image placeholder.
- **Informational Section**: Displays a centered heading and four cards, each with an illustration placeholder and subtext.
- **Testimonial/Quote Section**: Shows an inspiring quote with an attribution.
- **Call-to-Action Section**: Encourages users to sign up with a heading, subtext, and a contrasting button.
- **Footer**: Contains centered copyright text.

## Features

- **Responsive Layout**: Uses Flexbox to align and space elements.
- **Clean Design**: Emphasis on contrasting colors and clear typography.
- **Modular Structure**: Separated HTML sections for easy customization.
- **Google Fonts**: Incorporates the Roboto font for a modern look.

## Technologies Used

- **HTML5**
- **CSS3**
- **Google Fonts (Roboto)**

## Design Details

- **Color Palette:**
  - Dark Navy Blue (Header, Footer, Hero Background): `#1F2937`
  - Blue (Buttons, Informational Card Border): `#3882F6`
  - Light Gray (Subtext, Quote Background): `#E5E7EB`
  - Light White (Primary Text): `#F9FAF8`

- **Typography:**
  - **Font:** Roboto (imported via Google Fonts)
  - **Hero Main Text:** 48px, font-weight 900
  - **Subtext and Navigation Links:** 18px

- **Layout:**
  - Uses Flexbox for alignment in the header, hero section, and informational cards.
  - A `.container` class is used to center content and set a maximum width.
